# 🐍 Plataforma de Aprendizaje — Programación Orientada a Objetos
Esta aplicación web interactiva y gamificada ha sido desarrollada como componente práctico para la asignatura de **Programacion Orientada a Objetos**. Su objetivo es transformar el compendio teórico de Programación Orientada a Objetos (POO) en un entorno de aprendizaje dinámico inspirado en metodologías ágiles de estudio (estilo Duolingo).

La plataforma cuenta con **19 bloques de aprendizaje** divididos en tres niveles (Principiante, Intermedio y Avanzado), desafíos prácticos con análisis de código simulado y cuestionarios interactivos con acumulación de experiencia (XP) persistente.

## Requisitos Previos e Instalación

El proyecto incluye un archivo `requirements.txt` que actúa como la "lista de compras" automatizada de las dependencias necesarias.

### Pasos para instalar en una computadora nueva:
1. Asegúrate de tener Python instalado.
2. Abre la terminal o consola de comandos en la carpeta raíz del proyecto.
3. Ejecuta el comando para instalar todas las dependencias de golpe:
   ```bash
   pip install -r requirements.txt
